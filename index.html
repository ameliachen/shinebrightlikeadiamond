Basic layout

When you visit the app at: https://github.com/Freedom89/data_products

  1. Upload the data set
  2. You can choose the y-variable you want to predict. it is best to use the first column as the y-variable as that will be the default when it is updated
  3. You can select which variable to be factor variable based on the col names

Next Step

  1. As stated, mtcars is used as an example and the data is avaliable in github
  2. The dashboard auto computes the AIC and gives the optimal regression model

features

  1. Quickview to see the predict data plus the predicted value
  2. Easy to use - just have a csv file for training and one that you want to perform prediction
  3. Download button avaliable.
